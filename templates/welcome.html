Default Policy MO Diff Checker

Usage :

1. SSH to savbu-aut-srv9 with CEC Credentials

2. Go to dir /auto/ucs-india/defaultDiff

3. Create xml files for the builds/releases you want the Diff to be done.
Ex: touch granda.xml granada-mr1.xml ;

4. the content of the xml file should be fetched from autoconfig.xml of the corresponding release build url
Ex: For Granada-MR1 #183 build autoconfig xml - go to
https://cspg-releng.cisco.com/ucs-b-series/granada_mr1_sam-builds/160610-100230-rev280608-183c/perfocarta/model/ specific/mo/compute/autoconfig.xml
Copy entire xml except first line and save in in granada-mr1.xml you created in step 3

5. That's it just give the names of the xml files in below form
ex: Name of First Release: granada.xml, Name of Second Release: granada-mr1.xml
